Personal projects involving machine learning experiments, demonstration of practical skills in data analysis, model building, and evaluation.

* The file 'detection of brain tumors.ipynb' contains an implementation of a neural network architecture for detecting brain tumors based on MRI images
* The file 'classification.ipynb' contains the implementation of the image classification model pipeline (EuroSAT dataset)
* The file 'movie rating prediction.ipynb' contains a neural network development for predicting a movie's rating based on its characteristics
* The file 'termination of a client's cooperation with the bank.ipynb' contains an implementation of a model for predicting the probability of a client's termination of cooperation with the bank.
